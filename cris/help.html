Frequently Asked Questions

  • Q: Where can I access raw data files?
    A: Visit the Raw Data section for instructions.

  • Q: What RNA interactions are cataloged in CRIS?
    A: CRIS focuses on RNA-RNA interactions. RNA-protein interaction datasets are under development.

  • Q: How does CRIS ensure data quality?
    A: CRIS applies rigorous and transparent processing protocols to maximize accuracy. See the Statistics page for details.

  • Q: Will X for Y be available in CRIS?
    A: Dataset updates depend on project priorities. Contact Dr. Zhipeng Lu for inquiries.

  • Q: Can I collaborate or suggest improvements?
    A: Yes. Reach out to Dr. Zhipeng Lu to share your ideas.

  • Q: How should I cite CRIS in my research?
    A: Refer to the publication and cite according to the desired citation format.

  • Q: Can I submit my data to CRIS?
    A: User-submitted data is not currently supported. Future updates will include a submission portal for significant datasets.

  • Q: Can I use CRIS to analyze RNA interactions in disease?
    A: If an RNA-related structure or mechanism is implicated in a disease, CRIS data may provide insights.

Common Problems - Data Analysis

Most problems that occur during data analysis are a direct result of incorrect paths (filenames) and dependencies (software versions).
Please refer to the user manuals of the corresponding tool if available.

Configuring samtools

  • Load the conda module.
    module load conda
  • Create a new conda environment.
    conda create -n samtools_env ncurses zlib
  • Install required packages.
    conda install ncurses zlib
  • Install bzip2.
    conda install bzip2
  • Activate the environment.
    conda activate samtools_env
  • Set environment variables.
    export CPPFLAGS="-I$CONDA_PREFIX/include" && export LDFLAGS="-L$CONDA_PREFIX/lib"
  • Configure and install samtools.
    ./configure --prefix=PATH/bin/samtools --with-ncurses=$CONDA_PREFIX --with-zlib=$CONDA_PREFIX
  • Execute the program.
    make && make install
