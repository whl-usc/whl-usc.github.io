Glossary of Terms

Alignment
The process of arranging sequences of DNA, RNA, or proteins to identify regions of similarity or difference, typically by aligning reads from a sequencing experiment to a reference genome or transcriptome.
Annotation
The process of identifying and labeling features within a genome or transcriptome, such as genes, exons, introns, and other functional elements. It provides biological context to the sequence data.
BAM
A binary file format used for storing aligned sequences. BAM files are the compressed version of SAM files (see SAM below), which store information about the alignment of sequencing reads to a reference genome.
CLI (Command Line Interface)
A text-based interface used to interact with software applications or operating systems by typing commands. It is often used for running bioinformatics tools and pipelines.
DG
Duplex group - Refers to a group of sequences or reads that originate from the same duplex or RNA molecule, used in the context of RNA-seq to account for technical artifacts and improve data accuracy.
Gene Expression
The process by which information from a gene is used to synthesize gene products, typically proteins or RNA. Measurement of gene expression levels provides insights into cellular function and regulation.
Mapping
The process of aligning sequencing reads to a reference genome or transcriptome to determine their origin. Mapping helps in identifying where reads originate from and how they relate to known features.
Normalization
Techniques used to adjust raw data to account for technical variations and biases, such as differences in sequencing depth or sample processing. Normalization ensures that comparisons between samples are accurate and meaningful.
Pipeline
A series of computational steps or tools organized to process and analyze data from start to finish. In bioinformatics, pipelines automate and streamline complex workflows, from raw data processing to final analysis.
Preprocessing
Initial steps in data analysis that prepare raw data for further analysis, including quality control, filtering, and normalization. Preprocessing helps in cleaning and organizing data for accurate results.
Read
A single sequence of nucleotides obtained from a sequencing experiment. Reads are the fundamental data units generated by sequencing technologies and are used for alignment and analysis.
SAM
Sequence Alignment/Map. A text-based file format that stores aligned sequences of DNA or RNA reads. SAM files include information on read alignments, quality scores, and other metadata. BAM files are the binary version of SAM files.
STAR
A widely used aligner for RNA-seq data that performs fast and accurate alignment of reads to a reference genome, including the handling of spliced alignments.
Transcriptomics
The study of the transcriptome, which encompasses all RNA molecules in a cell or tissue at a specific time. Transcriptomics provides insights into gene expression patterns, regulatory mechanisms, and cellular responses.
