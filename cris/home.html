CRIS

The central resource hub dedicated to RNA-crosslinking methodologies and research.

Introducing the Crosslinking-based RNA Interactomes and Structurome (CRIS) database, a comprehensive resource advancing RNA research. CRIS consolidates data from multiple pipelines, offering a user-friendly platform for researchers with varying computational expertise. It integrates data from established pipelines like Cross-linked RNA Secondary Structure Analysis using Network Techniques (CRSSANT), ensuring consistent preprocessing for accurate analysis. CRIS facilitates analyses across datasets, uncovering patterns in RNA structures, gene expression, RNA-RNA interactions, and providing insights into RNA dynamics and conformations.

Key features include detailed guidelines for data processing workflows, access to raw and processed data, statistical information, visualization tools, and GitHub-linked scripts.

Trimmomatic

Trimmomatic is first used to remove the 3'-end adapters.

java -jar trimmomatic-0.36.jar SE -threads 16 -phred33 $fastq $Outprefix'_trim3.fastq' ILLUMINACLIP:$fasta:3:20:10 SLIDINGWINDOW:4:20 MINLEN:18

STAR

The mapping of pre-processed data is standardized from PARIS- and SHARC-based sequencing. STAR and various scripts from the CRSSANT pipeline are used.

/STAR-2.7.1a/bin/Linux_x86_64/STAR
--runThreadN 8
--runMode alignReads
--genomeDir $StaridxPath
--readFilesIn $Fastq
--outFileNamePrefix $Outprefix'_1_'
--genomeLoad NoSharedMemory
--outReadsUnmapped Fastx
--outFilterMultimapNmax 10
--outFilterScoreMinOverLread 0
--outSAMattributes All
--outSAMtype BAM Unsorted SortedByCoordinate
--alignIntronMin 1
--scoreGap 0
--scoreGapNoncan 0
--scoreGapGCAG 0
--scoreGapATAC 0
--scoreGenomicLengthLog2scale -1
--chimOutType WithinBAM HardClip
--chimSegmentMin 5
--chimJunctionOverhangMin 5
--chimScoreJunctionNonGTAG 0
--chimScoreDropMax 80
--chimNonchimScoreDropMin 20

Raw Data

Option 1. Bulk download data from a whole project via NCBI GEO.

    sra_grab.sh enables simple download and extraction of data from a GEO accession number.
sra_grab.sh

Option 2. Inividual sample (direct) download through NCBI GEO.

    At the bottom of GEO Accession Display, select SRA Run Selector.
    In the run table, select the SRRXXXXX file in the Run column that corresponds to the correct data.
    Under the FASTA/FASTQ download tab, click to download directly.
    To download to a server, copy the link that opens in your browser upon clicking the FASTQ link; use wget or curl.
    WARNING: Always check to see if a study uses paired-end or single-end sequences. If downloading through Option 2, be sure to download both files of the pair.
